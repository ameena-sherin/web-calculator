# 🧮 Calculator Project

## 📌 Description
This is a simple calculator project built using Python.  
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## 🚀 Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## 🛠️ Technologies Used
- Python
